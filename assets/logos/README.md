# Logo assets

Approved Red Door Collaborative logo files for use in HTML deliverables.

## Canonical source

All approved logo files live in Box:
> **Red Door File Database → Red Door Guidelines and Assets → RDC 2025 Branding**

Always pull from Box when preparing print materials. Files in this folder mirror the approved digital versions for HTML/web use.

## File naming convention

| File | Description |
|------|-------------|
| `rdc-logo-primary.svg` | Primary horizontal lockup — use on light backgrounds |
| `rdc-logo-primary-white.svg` | White version — use on dark or colored backgrounds |
| `rdc-logo-mark.svg` | Icon mark only — use when full lockup doesn't fit |
| `rdc-logo-mark-white.svg` | White icon mark |
| `rdc-logo-primary.png` | PNG fallback — 2x (retina) |
| `rdc-logo-primary-white.png` | White PNG fallback — 2x |

## Usage rules

- Minimum digital size: **85px wide**
- Minimum print size: **0.25 in wide**
- Clear space: equal to the logo's bounding square on all four sides
- Do not stretch, rotate, skew, recolor, or add drop shadows
- Do not place on backgrounds that reduce contrast below WCAG AA

## Adding files to this folder

Drop approved SVG and PNG exports here. Follow the naming convention above. PNG files should be exported at 2x resolution minimum.
